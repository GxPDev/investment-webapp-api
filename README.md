Investment WebApp API

ASP.NET Core backend for an investment management platform featuring authentication, transaction management, dashboard services, and real-time communication architecture.

🚀 Features

* User authentication system
* Investment management logic
* Dashboard backend services
* Real-time communication using SignalR
* Database migrations
* MVC architecture
* Scalable service structure

🛠️ Tech Stack

* ASP.NET Core
* C#
* Entity Framework Core
* SignalR
* SQLite
* MVC Architecture

📂 Project Structure

* Controllers
* Services
* Models
* Data
* Migrations
* Hubs
* Views

📈 Purpose

This project was built as part of a full-stack investment platform architecture to demonstrate backend system design, API development, and scalable ASP.NET application structure.
